Software

Created a Mock Car Dealership Application utilizing ASP.NET MVC and a SQL Server database. C#/.NET combined with ADO.NET to access data. HTML, JavaScript, and Bootstrap were used for the front-end. A user can register to be an Administrator or Salesperson. Admin can modify cars and car dealership details. Salespeople create purchase logs and make sales to customers. I practiced test-driven development with NUnit framework to effectively plan and refactor my code with ease.
